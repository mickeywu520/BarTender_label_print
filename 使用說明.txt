標籤列印管理程式 - 使用說明
=====================================

程式功能：
- 載入 BarTender BTW 檔案
- 從 ERP 系統 API 獲取進貨單資料
- 自動填入標籤欄位
- 列印標籤到指定印表機

使用步驟：
1. 選擇 BTW 檔案
   - 點擊「瀏覽...」按鈕
   - 選擇要使用的 BarTender 標籤檔案 (.btw)

2. 輸入進貨單資訊
   - 進貨單別：輸入單據類型 (3開頭=一般進貨，5開頭=托外進貨)
   - 進貨單號：輸入單據號碼
   - 進貨項次：輸入項次序號

3. 獲取資料
   - 點擊「獲取資料」按鈕
   - 系統會自動呼叫對應的 API 端點
   - 右側會顯示獲取到的資料

4. 設定列印參數
   - 數量：會自動填入，可手動修改
   - D/C：手動輸入
   - HW Ver：手動輸入硬體版本
   - FW Ver：手動輸入韌體版本

5. 列印標籤
   - 點擊「列印標籤」按鈕
   - 確認列印資訊
   - 執行列印

API 設定：
- 基礎 URL: http://192.168.0.13:100/api/ErpToBarcode/
- 一般進貨端點: getPurchaseReceipt
- 托外進貨端點: getSubcontractReceipt

印表機設定：
- 預設印表機 IP: 192.168.6.47

注意事項：
1. 需要安裝 BarTender SDK 才能正常使用列印功能
2. 確保網路能連接到 API 服務器
3. 目前 JSON 解析使用模擬資料，實際使用時需要安裝 Newtonsoft.Json 套件
4. BarTender 相關程式碼已註解，需要 SDK 後取消註解
