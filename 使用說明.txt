標籤列印管理程式 - 使用說明
=====================================

程式功能：
- 載入 BarTender BTW 檔案
- 從 ERP 系統 API 獲取進貨單資料
- 自動填入標籤欄位
- 智能預覽尺寸修正
- 高品質標籤預覽
- 列印標籤到指定印表機

使用步驟：
1. 選擇 BTW 檔案
   - 點擊「瀏覽...」按鈕
   - 選擇要使用的 BarTender 標籤檔案 (.btw)

2. 輸入進貨單資訊
   - 進貨單別：手動選擇單據類型 (一般進貨 或 托外進貨)
   - 進貨單號：輸入單據號碼
   - 進貨項次：輸入項次序號

3. 獲取資料
   - 點擊「獲取資料」按鈕
   - 系統會自動呼叫對應的 API 端點
   - 右側會顯示獲取到的資料

4. 設定列印參數
   - 數量：會自動填入，可手動修改
   - D/C：手動輸入
   - HW Ver：手動輸入硬體版本
   - FW Ver：手動輸入韌體版本

5. 列印標籤
   - 點擊「列印標籤」按鈕
   - 確認列印資訊
   - 執行列印

6. 預覽功能
   - 點擊「更新預覽」按鈕可查看標籤預覽
   - 預覽會自動調整尺寸確保顯示正確
   - 列印後如預覽異常會自動修正

API 設定：
- 基礎 URL: http://192.168.0.13:100/api/ErpToBarcode/
- 一般進貨端點: getPurchaseReceipt
- 托外進貨端點: getSubcontractReceipt

印表機設定：
- 測試印表機 IP: 192.168.0.240 (印表機名稱: LC01)

注意事項：
1. 需要安裝 BarTender SDK 才能正常使用列印功能
2. 確保網路能連接到 API 服務器 (192.168.0.13:100)
3. 已整合 Newtonsoft.Json 套件，支援完整 JSON 解析
4. 已啟用 BarTender SDK 功能，支援真實標籤預覽和列印
5. 預覽圖片會自動儲存在程式目錄的 PreviewTemp 資料夾
6. 列印後預覽如有異常會自動重新載入 BTW 檔案修正
